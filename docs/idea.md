1. Definición de la Idea
Vellum Canvas es un motor de gestión y narrativa para juegos de rol de mesa (TTRPG). La aplicación permite a un Director de Juego (DM) crear un entorno visual interactivo donde puede gestionar mapas, escenas narrativas y una economía dinámica, mientras los jugadores interactúan con sus personajes y el mundo en tiempo real. El sistema combina la lógica de un gestor de tareas con la estética de una visual novel.

2. Problema que resuelve
Actualmente, las sesiones de rol sufren de una fragmentación de herramientas: los DMs suelen alternar entre mapas estáticos, hojas de cálculo para inventarios, notas de texto y sistemas de chat. Vellum Canvas centraliza la experiencia, permitiendo que la narrativa visual y la gestión de datos (oro, ítems, misiones) ocurran en una misma plataforma integrada, reduciendo la carga cognitiva del DM y aumentando la inmersión del jugador.

3. Usuario Objetivo
Directores de Juego (DM): Que buscan una herramienta profesional para automatizar la gestión de sus mundos y presentar contenido visual atractivo.

Jugadores de Rol: Que desean tener un control claro de su progresión, inventario y posición en el mundo de juego.

4. Funcionalidades Principales (MVP)
Gestión de Usuarios (Roles): Sistema de permisos para diferenciar al DM (Root) de los jugadores.

Cartografía Interactiva: Mapa central con iconos de personajes arrastrables (Drag & Drop) y puntos de interés (Pins) vinculados a eventos.

Motor de Escenas Visuales: Interfaz estilo Visual Novel para diálogos, donde el DM configura fondos, personajes y textos.

Tablón de Misiones (Gremio): Sistema de gestión de misiones con rangos (E a S) y recompensas automáticas.

Códice de Objetos: Herramienta para crear una base de datos de objetos con nombres, descripciones y estadísticas.

5. Funcionalidades Opcionales
Plantilla de Tienda: Escenas especiales donde el DM configura un inventario de venta y los jugadores realizan transacciones automáticas.

Habitaciones Privadas: Escenarios dedicados para que los jugadores gestionen su equipamiento personal de forma privada.

Sistema de Rangos de Gremio: Progresión del nivel de gremio basada en la experiencia acumulada por misiones completadas.

6. Mejoras Futuras
Sistema de Combate Integrado: Gestión de iniciativa, vida y estados de combate sobre el mapa.

Multiverso de Mapas: Capacidad de gestionar múltiples mapas y regiones interconectadas simultáneamente.

Librería de Assets en la Nube: Integración con servicios de almacenamiento para que el DM pueda cargar y organizar sus propias galerías de imágenes.
