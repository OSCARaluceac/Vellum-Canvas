# 🛡️ Vellum Canvas: RPG Master Engine

**Vellum Canvas** es un motor de gestión y narrativa visual para juegos de rol de mesa (TTRPG). Permite a los Directores de Juego (DM) orquestar sesiones inmersivas mediante mapas interactivos, sistemas de escenas dinámicas y una economía automatizada.

## 🚀 Tecnologías
| Frontend | Backend | Herramientas |
| :--- | :--- | :--- |
| React 18 + Vite | Node.js | TypeScript |
| Tailwind CSS | Express | Trello (Agile) |
| React Router | API REST | Git / GitHub |

## 🕹️ Funcionalidades Clave
- **Cartografía Interactiva:** Mapa con Drag & Drop para iconos de personajes.
- **Motor de Escenas:** Renderizado tipo Visual Novel para diálogos y eventos.
- **El Códice:** Base de datos para la creación y gestión de objetos.
- **Gremio-Taskflow:** Tablón de misiones con recompensas automáticas.
- **Economía Viva:** Gestión de inventarios, tiendas y equipamiento.
